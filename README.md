# dotfiles

## Installation

## Linux / Mac

```sh
# curl https://setup-shell.vercel.app | sh -
sh install.sh
```

## Windows

```sh
git clone https://github.com/hasez/dotfiles.git ~/src/github.com/hasez/dotfiles
cd ~/src/github.com/hasez/dotfiles
powershell -Command Start-Process -Verb runas install.bat
```
